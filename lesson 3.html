Lesson 3: Display Property

The display property in CSS controls how an element is displayed.

Block Level Element

Block level elements take up any possible space by default. Every element at the block level will begin a new line on the page, stacking the page downward. Apart from vertical stacking, Additionally, block level elements will occupy the same amount of horizontal space. as much as possible.

Examples:

  • <p>
  • <ol>, <ul>, <dl>
  • All headings
  • <articles>, <section>, <div>

Inline Level Element

Inline level elements are displayed in a line, and they do not force a new line on the text after them. Even if there are several links in a row, it will just be one line.

Examples:

  • <a>
  • <strong>, <em>, <b>, <i>, <q>, <mark>
  • <span>

Div Tag

With this tag, content on a webpage would be divided into different sections. It adds structure, and allows you to apply styles to individual sections of the web page.

Span Tag

Used to give emphasis to words in a sentence, and adds structure to a webpage.

This lesson helped me learn that there are other ways to help display your text on an HTML element, and not just one singular bland way.
