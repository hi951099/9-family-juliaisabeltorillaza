Lesson 4: Classes

Class is a user-defined selector, and is used to apply style rules to individual HTML tags. It is reusable, and allows you to set the same formatting rules to multiple elements. They are defined by periods (.)

Two methods of writing class.

Indicate the HTML element then a period, then the class name.
Example: p.albedo{color:purple} - This can be used on paragpraphs.
Put a period then the class name.
Example: .albedo{color:purple} - This can be used on an individual HTML element.

Whats the point of using element and class selectors?

Class Selector (.classname)
- reusable for multiple elements
Element + Class selector (p.classname)
- Focuses on only one element

In this lesson, I learned how useful CSS classes are in styling web pages. Classes allow us to apply the same style to multiple elements, making it easier to keep a consistent design across the page.
