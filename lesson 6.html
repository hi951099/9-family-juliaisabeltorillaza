Lesson 6: Setting Dimensions

Dimensions can be managed with CSS, including HTML elements' visibility, width, or height

Property for Dimensions

Visibility
- Shows if an element is visible or hidden
- Example: visibility:visible - The value is visible
- Example: visibility:hidden - The value is hidden
Width
- Shows the width
- Example: width:100px
Height
- Shows the height
- Example:height:100px
Line-Height
- Shows the line spacing
- Example:line-height:11px
Max-Height
- Shows the maximum height
- Example: max-height:25px
Min-Height
- Shows the minimum height
- Example: min-height: 11x
Max-Width
- Shows the max width
- Example: max-width:13px
Min-Width
- Shows the min width
- Example: min-width: 9px

Because of this lesson, I learned how to properly give dimensions to any of my html webpages.
