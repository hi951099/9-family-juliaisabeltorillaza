Lesson 2: Types of CSS

A CSS rule has a few parts: The selector is what you're styling (like a heading or a paragraph). The property is what you're changing (like color or font-size). The value is what you want it to be (like blue or 16px). Together, a property and value make a declaration. A bunch of declarations in curly brackets {} is called a declaration block.

The 3 types of CSS:

External CSS
You link a separate .css file to your HTML. This is best for keeping everything organized and making the whole site match the same style.
Embedded/Internal CSS
This goes inside a style tag in the head of your HTML. Youâ€™d use this if you want just one page to look different.
Inline CSS
This is when you put the style directly in the HTML tag using the style="" attribute. It only changes that one thing. It's kind of messy, though, so only use it if you really have to.

I learned that there are different types of CSS, all of which have different uses. They all design your html website, but they all have distinct ways to design them.
